# BlockChain Based Voting Dapp

## How to set Up
	1. run 'npm install'
	2. run 'npm run ganache'
	3. run 'npm start'

## How to run
    1. Open http://127.0.0.1/3000
    2. Click on any name to Vote

## How to change candidates list
    1. Navigate to **candidates.js** and add or remove the names in the array.
